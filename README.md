# Secure Pass

A JavaFX-based Password Strength Analyzer built using Java, JavaFX, MySQL, JDBC, Maven, and SHA-256 hashing.

## Features

- Password Strength Analysis
- Entropy Calculation
- Password Suggestions
- Password Generator
- Show/Hide Password
- SHA-256 Password Hashing
- MySQL Database Integration
- Records Dashboard
- Statistics Dashboard
- JavaFX GUI
- CSS Styling

## Tech Stack

- Java 22
- JavaFX
- Maven
- MySQL
- JDBC
